# Market Direction 📈

A market analytics dashboard built using React, TypeScript, and Vite to analyze market trends, backtesting strategies, risk, and portfolio performance.

## Features
- Market trend visualization
- Backtesting of strategies
- Risk analysis
- Portfolio overview
- Classification results

## Tech Stack
- React
- TypeScript
- Vite
- CSS

## How to Run
1. Clone the repository  
   git clone https://github.com/Mithrasree25/Market-Direction.git

2. Install dependencies  
   npm install

3. Start the project  
   npm run dev
